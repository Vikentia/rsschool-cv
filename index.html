Condition:

Write a function to convert a name into initials. The output should be two capital letters with a dot separating them. It should look like this: Sam Harris => S.H

Solution:

function abbrevName(name) {
    return name.split(' ').map(item =>item[0]).join('.').toUpperCase()
}
console.log(abbrevName("Sam Harris"));
